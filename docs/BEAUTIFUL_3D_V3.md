# AORI ROOM Beautiful 3D v3

このリリースでは、タイトルとルーム内のキャラクターを同じThree.js製オリジナル3Dアバターへ統一します。

## 表現

- 肌、髪、布、金属を分けた物理ベース素材とトゥーン輪郭
- 立体的な瞳、虹彩、まつ毛、眉、頬、口元
- ルミ／ミオ／セナで異なる髪型、アクセサリー、衣装ディテール
- レイヤードスカート、靴、胸飾り、ラケット面とストリング
- まばたき、呼吸、視線、髪・衣装の追従揺れ
- 選択中キャラクターをタイトル上でリアルタイム3D描画
- WebGL不可時は既存CSSキャラクターを自動フォールバック

## 検証契約

`release.json`の`visualVersion`は`beautiful-3d-v3`です。品質ゲートはiPhone 13相当のWebKitで`.title-preview--ready`を待ち、実際の3D描画スクリーンショットを保存します。ルームAPI、Durable Object、WebSocket、キャラクター同期、再接続の実通信試験も従来どおり必須です。

初回のWebKit画像を目視確認し、タイトル用カメラに背を向けていた回転を修正しました。再検証では顔、瞳、衣装正面、ラケットを含む正面表示を合格条件とします。
